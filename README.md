# 2 Truths And A Lie

2 Truths And A Lie team game implemented using Python/Fast API and React.

## Linter

Install flake8:

```
pip install flake8
```

Run flake8:

```
flake8
```

## Run tests

Install Pytest:

```
pip install pytest
```

```
pytest -v
```

## Run locally

```
docker compose up
```

Access frontend at [http://localhost:3000](http://localhost:3000)
